常見問題集

Acronis Cyber Protect Cloud 將網路安全、資料防護和安全性管理整合到單一的解決方案中。透過獲獎肯定的 Acronis Active Protection 技術保護客戶資料和端點，此技術已透過 AI 和行爲型反惡意軟體偵測增强。還透過裝置控制提供基本的資料遺失防護 (DLP)，防止資料洩漏。

此解決方案還透過弱點和安全漏洞評估提供端點防護的完整可見性。

Advanced Security 藉由 URL 篩選和攻擊預防擴展網路安全解決方案的功能，以抵禦包括 Web 型攻擊和惡意探索試圖等更多威脅。還藉由增强的病毒碼簽章資料庫，提高對已知惡意軟體偵測的速度和準確性。附加元件套件允許在 Acronis Cloud 中對已備份的資料進行更積極的惡意軟體掃描，從而阻止威脅反覆發生。

透過 Advanced Security，服務供應商還可以更高效地對威脅進行應變。Acronis Cyber Protection Operation Center (CPOC) 會監控威脅態勢，並針對新與威脅發佈警示。防護計畫可根據新與威脅警示自動調節，從而導致發生更頻繁備份或更積極掃描等情況。

搭配 Advanced Security 的 Acronis Cyber Protect Cloud 包含基準產品之獲獎肯定的防勒索軟體防護技術。還以端點、伺服器、網路共用和備份中的客戶資料為目標，讓服務供應商針對勒索軟體和加密挖礦程序提供無可比擬的即時防護。Acronis 的勒索軟體防護技術還包含熵分析，能更高效地覆蓋進階勒索軟體威脅。

讓我們的防勒索軟體技術真正獨一無二的是，它在幾秒内自動復原任何受影響的加密資料，確保無客戶資料遺失。

Advanced Security 藉由進一步改進偵測速度和準確性的增强病毒碼簽章資料庫資料庫，改善了 Acronis Cyber Protect Cloud 的 AI 和行爲型反惡意軟體偵測。

進階端點防護功能還覆蓋包括 Web 型攻擊和軟體惡意探索試圖在内更大範圍的威脅，確保了全面有效的防護。

藉由 Advanced Security 附加元件，服務供應商還可以在 Acronis Cloud 中卸載客戶的端點並提供更積極的掃描，確保已備份的資料無惡意軟體。復原流程同樣地受反惡意軟體掃描和防病毒定義檔更新保護。

Acronis Cyber Protection Operation Center (CPOC) 會監控威脅態勢，並針對惡意軟體、弱點或自然災害等新與威脅發佈警示。智慧防護計畫可自動調整為主動涵蓋這些威脅，大大改善了反應時間。
